Real-Time Product Secction


Sección de productos actualizada en tiempo real hecho en ASP.NET MVC5.

ASP.NET Entity Framework DataTable BootBox

Tecnologías Utilizadas

  • MVC5
  • DataTables
  • BootBox
  • Entity Framework

Descripción del funcionamiento

Sección de productos en tiempo real creada con ASP.NET MVC5 y API creada con funcionalidades CRUD.

La sección de proyectos se actualiza con cada acción realizada; la información es mostrada utilizando DataTables y las alertas son mostradas con BootBox.

API Endpoints

La aplicación utiliza una web API creada en MVC5, cuenta con controladores para Crear, Leer, Actualizar y Eliminar. El formato de salida puede ser XML o JSON.

  • GET: /api/producto
  • GET: /api/product/1
  • POST: /api/product
  • PUT: /api/product
  • DELETE: /api/producto
