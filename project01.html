Twitter Bot utilizando Tweepy


Bot hecho en Python, este bot genera una imagen con el pronóstico del tiempo de una ciudad ingresada por el usuario a través de un Tweet.

Python TweePy NumPy

Librerías necesarias.

Para realizar este proyecto se utilizaron múltiples librerías, las principales son:

  • PIL
  • NumPy
  • TweePy

APIs utilizadas.

Para poder acceder a la información del pronostico del tiempo, es necesario crear una cuenta en OpenWeather.

Para poder utilizar los servicios de Twitter es necesario aplicar a una cuenta de desarrollador.

Descripción de funcionamiento

  1. Cuando el usuario menciona la cuenta del bot en Twitter se obtiene el mensaje.
  2. Luego se verifica que la cadena del tweet corresponda a una ciudad en Guatemala.
  3. Si es una ciudad se genera la imagen con los datos brindados por el API de OpenWeather.
  4. Una vez generada la imagen se procede a mencionar al usuario que solicito la información del clima.
